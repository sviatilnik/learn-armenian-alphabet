# Тренажёр армянского алфавита

CLI-приложение на Go для интерактивного изучения армянского алфавита: два режима игры, статистика ходов и обратная связь по ответам.

## Цель

Помочь запомнить армянские буквы и их названия/звуки: программа показывает случайную букву, пользователь вводит ответ, получает результат и видит накопленную статистику. Игра идёт до прерывания (Ctrl+C).

## Реализация

- **Язык:** Go 1.25+
- **Структура:**
  - `cmd/game/` — точка входа, меню, обработка сигналов и запуск игры
  - `internal/alphabet` — словарь букв (руна, название, звук), выбор случайной буквы
  - `internal/game` — интерфейс `Game`, режимы «Назвать букву» (`LetterGame`) и «Назвать звук» (`SoundGame`), игровой цикл
  - `internal/statistic` — структура хода (`Stat`), интерфейсы `Recorder`/`Tracker`, сбор и вывод статистики

- **Режимы:**
  1. **Назвать букву** — показывается символ (Ա/ա и т.д.), ввод названия буквы (например, «айб»).
  2. **Назвать звук** — показывается символ, ввод звука/транслитерации (например, «а»).

- **Игровой цикл:** случайная буква и регистр → вывод «Буква: X», приглашение «Ваш ответ:» → чтение строки → сравнение с эталоном → сообщение «Правильно!» / «Неправильно. Правильный ответ: …» → запись в статистику → вывод блока статистики → следующий вопрос.

- **Завершение:** по Ctrl+C (SIGINT/SIGTERM) выводится прощание и при наличии — итоговая статистика.

## Запуск

### Требования

- Установленный [Go](https://go.dev/dl/) (1.25 или новее).

### Сборка и запуск через Make

```bash
# Сборка бинарника (по умолчанию в ./bin/game)
make build

# Запуск игры (собирает при необходимости)
make run

# Очистка собранного бинарника
make clean
```

### Без Make

```bash
# Сборка
go build -o bin/game ./cmd/game

# Запуск
./bin/game
```

Или запуск без сохранения бинарника:

```bash
go run ./cmd/game
```

После запуска выберите режим (1 или 2), вводите ответы по буквам и завершайте игру по Ctrl+C.
